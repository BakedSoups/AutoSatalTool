Automatic Students Assessing Teaching and Learning (SATAL) Tool 
==================== 
- Create slides by just inputing A name
- Automate data cleaning with GPT 
- Automate data analysis with GPT

## Installation
```bash
pip install python-pptx
pip install qrcode
```
## Usage
```bash
python main.py
```
